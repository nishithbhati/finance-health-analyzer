Financial Health Analyzer
A fully responsive, client-side web application designed to help users track expenses and visualize their financial health using the 50/30/20 budgeting rule.

This project demonstrates strong skills in Vanilla JavaScript, CSS3 styling (Glassmorphism), and DOM manipulation without external frameworks.

Features:

Smart Analysis: Implements the 50/30/20 budget rule (Needs, Wants, Savings).
Data Persistence: Uses localStorage to save user data even after the browser is closed.
Reactive UI: Real-time updates of balance and progress bars.
Modern Design: Glassmorphism aesthetic with smooth CSS animations.

Tech Stack:
HTML5: Semantic structure and accessibility.
CSS3: Flexbox, CSS Variables, and Backdrop Filters.
JavaScript (ES6+): Arrow functions, Array methods, DOM Manipulation, and LocalStorage API.
